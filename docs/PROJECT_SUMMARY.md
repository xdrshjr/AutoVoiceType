## 📊 项目概览

### 项目定位

AutoVoiceType 是一款专为 Windows 平台设计的智能语音输入工具，通过全局快捷键实现"按住即说"的无缝语音输入体验。

### 核心价值

- 🚀 **提升效率**: 语音输入速度远超打字
- ✋ **解放双手**: 减轻长时间打字疲劳
- 🌐 **全局可用**: 支持99%的Windows应用
- 🎯 **简单易用**: 一键触发，自动输入
- 💡 **智能识别**: 基于阿里云的高精度识别
